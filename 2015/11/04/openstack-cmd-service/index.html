配置 stack 用户

创建 stack 用户+

sudo groupadd stack
sudo useradd -g stack -s /bin/bash -d /opt/stack -m stack
添加 stack 用户权限。
sudo echo “stack ALL=(ALL) NOPASSWD: ALL” >> /etc/sudoers
切换到 stack 用户
sudo su - stack
