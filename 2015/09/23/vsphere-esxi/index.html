vsphere系列 嵌套虚拟化

发表于 2015-09-23   |   分类于 vsphere   |     |   次阅读

虚拟化嵌套：顾名思义就是在虚拟化环境中在嵌套一层虚拟化。

  • 虚拟机-编辑设置-选项-常规选项。版本选择其他。

  • 虚拟机-虚拟硬件-CPU-选项-CPU/MMU虚拟化-选择“硬件CPU和MMU”。

  • 虚拟机-编辑设置-虚拟硬件-CPU-硬件虚拟化-勾选“向客户机操作系统公开硬件辅助的虚拟化”。

  • 确认虚拟机版本为最新，esxi 5.5u2版本是vm10，如果不是vm9.升级虚拟机兼容性。

  • 点击宿主ESXI主机-右边配置-网络-属性，找到你ESXI虚拟机连接的端口组，点击编辑，在安全选项中，把混杂模式、伪传输勾选，选择接受！

否则会出现在嵌套环境下的虚拟机操作系统安装完成后，IP能获取，但是上不了外网。
